Learning to use FICS - Freechess.org

The Free Internet Chess Server - FICS - is the place where I learnt to play chess online. It will always be free to play and use all the features, which are many. However this is not always obvious to newcomers. FICS is full of great players, people and training - if you know how to use it. This post aims to explain how to get started on FICS.

Getting Started

Because FICS is a server and not a website like Chess.com or Lichess.org - you need a program to use it.

Depending on which operating system you have, there are different programs.

Programs

I’ve preferred three programs.

Babas Chess

Babas is GUI driven with tons of options. Install and run it and follow the prompts and you will end up a screen resembling the above. From here you can explore all sorts of options in File > Preferences or just hit the Quick Commands menu item and play some chess.

On FICS there is a channel 1 for general help and channel 6 for interface help where you can ask for help or just Google. Your mileage with Babas may vary but it is certainly popular.

If you wish to run Babaschess on Linux or Mac OS X - use the awesome Wine and turn off sounds and emoticons if they give you crashes.

Thief

Thief is dominant among Bughouse players and requires a bit more knowledge of FICS and commmands. However with a bit of experimenting and Googling it is simple enough to use. If you like the look of bughouse chess - which is awesome in real-life - check out Bughouse.net or channel 22 on FICS.

Thief is also Windows native but can run on Linux and Mac OS X using Wine.

Raptor

Raptor is a newer Java based interface written by members of FICS. It runs on all operating systems and is quite popular among Mac OS X and Linux users. However, it works just fine on Windows as well.

The configuration of Raptor can be extensive and a visit to the homepage linked above is good idea for further documentation. There are also reviews around the Web to see how others configure and use Raptor well. My setup is the screenshot above :)

Registering on FICS

The best getting started guide to FICS has already been written by the friendly Raptor guys. Download Raptor and follow along for screen by screen explanations!

Ongoing Learning

FICS is a great place to play the variants of chess; bughouse, atomic, losers, suicide. It is also a great community for strong blitz, tournaments in channel 49 and long time controls in Teamleague.

However I found it necessary to to use other sites such as Chesstempo, ChessPub to learn beyond the basics of playing chess on FICS.

Give them a shot!

Summary

The very first person I saw when I first joined FICS was the user TioHoracio. After observing and learning from his games I learnt that he regularly defeats International Masters(IM’s) in Over-The-Board(OTB) chess and on FICS. This is the way I percieve the quality of FICS - hidden but valuable.

FICS is a throwback to internet chess from years ago rather than the current internet. It is still a place where you can make friends, learn from experts for free and play long series' of games - something I found horribly lacking on non-free sites.
